# Chess Engine
The following project is a chess engine UI which was written in C++ and SFML. Future plans include engine integration.

## Overview
Built an interactive chess engine UI that contains a board representation that enforces rules such as movement of pieces, castling, promotion, en passant etc. Can compute the legal moves for the current position using movement constraints and bitboards. Contains many classes including a ChessBoard class, piece classes (Pawn, Knight, etc.) and a PGN class that can print out the PGN notation for the game being played. Includes a Move class that has a Move Node that can be used to build a game tree which can be used to produce the PGN.

## How to Run
How to Run: To run this, load the project in Visual Studio 2022 and run it.
